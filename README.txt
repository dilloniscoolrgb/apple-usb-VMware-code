yo if ur reading this then this is my macOS VMware USB

made by: Dillon

what this usb is for:
this usb has the files i use for my macOS virtual machine. if something breaks i can use these files to set it back up without downloading everything again.

stuff i use:
- VMware Workstation
- macOS ISO
- VM files
- backups

how i normally set it up:
1. plug in usb
2. copy files to pc
3. open VMware
4. load the vm or make a new one
5. select the macOS iso if needed
6. boot it up
7. hope nothing explodes

recommended specs:
- 8GB+ RAM
- 4 CPU cores
- lots of storage
- patience

important:
DO NOT DELETE RANDOM FILES
i probably put them there for a reason

if the vm stops working:
- check VMware settings
- check if virtualization is enabled
- make sure the iso is still there
- google the error before panicking
